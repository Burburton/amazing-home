# Amazing Home Demo Walkthrough

This document describes the key demo scenarios for creating a promotional video or GIF.

---

## Demo Flow (2-3 minute video)

### Scene 1: Welcome (5 seconds)
- Open application at https://burburton.github.io/amazing-home/
- Show WelcomePanel with "Get Started" guide
- Click "Try Demo Project" button

### Scene 2: Floor Plan Upload (15 seconds)
- Click "Upload Floor Plan"
- Select a sample floor plan image
- Show image loaded on 2D canvas
- Pan/zoom to show floor plan

### Scene 3: Wall Tracing (20 seconds)
- Click "Draw Walls" mode
- Trace walls from floor plan
- Show wall snapping and line drawing
- Or: Use "Detect Walls" experimental feature

### Scene 4: 3D Preview (10 seconds)
- Toggle to 3D view
- Show walls rendered as 3D boxes
- Orbit camera around model

### Scene 5: Furniture Placement (25 seconds)
- Back to 2D view
- Click furniture category (Sofa, Bed, etc.)
- Click to place furniture on floor plan
- Drag to reposition
- Rotate with rotation control
- Resize dimensions in inspector

### Scene 6: Layout Suggestions (15 seconds)
- Open Layout Suggestion panel
- Show AI-generated suggestions
- Click a suggestion card
- Show furniture auto-arranged

### Scene 7: Save & Export (10 seconds)
- Save layout version
- Export JSON file
- Show download in browser

### Scene 8: Custom Furniture (15 seconds)
- Show Custom Furniture form
- Create custom sofa with dimensions
- Place custom furniture
- Save as reusable asset

## Key Features to Highlight

- **Browser-based**: No install required
- **Real-time 3D**: See layout as 3D model
- **Furniture library**: 7 categories, custom assets
- **Layout suggestions**: AI-generated arrangements
- **Export/Import**: JSON persistence

---

## Screen Recording Tips

- Use 1920x1080 resolution
- Hide browser bookmarks/toolbar
- Use sample floor plan with clear walls
- Place furniture deliberately (not random)
- Show smooth transitions
